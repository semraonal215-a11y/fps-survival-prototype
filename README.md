# fps-survival-prototype
FPS Survival Prototype developed with Unity and C#

## Features

- FPS player controller
- Modular weapon system
- Ammo & reload mechanics
- Weapon switching
- NavMesh based enemy AI
- Wave-based enemy spawning
- Game Director system
- Object pooling for impact effects
- Minimap system using Render Texture
- Audio Manager system
- UI systems (Start Menu, HUD, Game Over)

## Technologies

- Unity
- C#
- Unity NavMesh
- Unity New Input System (Legacy compatible)

## Systems Included

- Player controller system
- Enemy AI system
- Weapon system
- Inventory system
- Audio manager
- Game flow manager
- Object pooling
- UI systems
